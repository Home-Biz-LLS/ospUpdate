# curriculum

Translations of curriculum codes to clear text descriptions

## Installing

`npm i -S @skolplattformen/embedded-api` or `yarn add @skolplattformen/embedded-api`

## Calling

```javascript
import parse from '@skolplattformen/curriculum'

parse('MU') // { code: 'MU', category: '', name: 'Musik' }
parse('M1SP') // { code: 'M1SP', category: 'Moderna spr책k, elevens val', name: 'Spanska' }
parse('M2TY') // { code: 'M2TY', category: 'Moderna spr책k, spr책kval', name: 'Tyska' }
parse('MLSMI') // { code: 'M2TY', category: 'Modersm책l', name: 'Samiska' }
```
